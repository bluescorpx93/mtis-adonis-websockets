### Adonis Tryout with Websockets

To Test This, 

- Install Package Dependencies
```sh
$ cd mtis-adonis-websockets
$ yarn install

# or, with npm
$ npm install
```

- Run App in PROD Mode
```sh
$ cd mtis-adonis-websockets
$ yarn start
```

- Run App in DEV Mode
```sh
$ cd mtis-adonis-websockets
$ adonis serve --dev
```
